# 兼容性视角 — Compatibility Perspective

## Evaluation Criteria

### Python Version Compatibility
- Works with Python 3.9+
- Works with Python 3.11, 3.12
- No deprecated API usage warnings

### Node Version Compatibility
- package.json specifies engine requirements
- Works with LTS Node versions
- No native module compatibility issues

### Browser Compatibility (Frontend)
- Works in Chrome, Firefox, Safari, Edge
- Progressive enhancement approach
- No browser-specific bugs

### API Version Compatibility
- Breaking changes are versioned
- Backward compatibility is maintained
- Deprecation warnings are clear
