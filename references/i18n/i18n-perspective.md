# 国际化视角 — Internationalization Perspective

## Evaluation Criteria

### No Hardcoded Strings
- All user-facing strings are externalized
- Message IDs are descriptive
- Pluralization is handled correctly

### Language Coverage
- Core languages are fully translated
- Translation completeness is tracked
- RTL languages are supported if needed

### Locale Formatting
- Dates use locale-appropriate format
- Numbers use correct separators
- Currency formatting is correct

### Translation Quality
- Translations are accurate
- No machine-translation artifacts
- Cultural appropriateness is considered
